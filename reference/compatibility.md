# Compatibility

Politica:
- versao menor pode esclarecer texto sem mudar campos obrigatorios
- versao maior pode alterar contrato
- consumidores devem validar campos obrigatorios
- campos desconhecidos podem ser ignorados quando nao forem criticos

Compatibilidade inicial:
- DEVJOB/1.0 gera DEVREPORT/1.0
- sem dependencia de runtime
- sem dependencia de bridge
- sem camada de transporte definida
